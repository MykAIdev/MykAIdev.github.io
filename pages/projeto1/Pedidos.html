💼 Projeto: Processamento de Pedidos com Validação e Integração

Automatize a gestão de pedidos com IA, RPA e integrações inteligentes.

🧩 O que este projeto faz?

  • 1. Lê e-mails com anexos (PDF/Excel)
  • 2. Extrai dados de cliente, itens e valores
  • 3. Valida os dados com base existente
  • 4. Registra o pedido em planilha
  • 5. Envia resposta automática (aceito/rejeitado)
  • 6. (Opcional) Integra com sistema web via RPA

🛠 Tecnologias Aplicadas

Power Automate

Fluxos na nuvem, integração com Outlook, Excel e respostas automáticas.

Blue Prism

Extração com Decipher, tratamento e inserção em sistemas legados.

Python

Processamento completo com imaplib, pandas, PyPDF, openpyxl, e-mails e Selenium.
